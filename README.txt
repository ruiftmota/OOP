
O projeto foi feito sem dependência de qualquer IDE. Os ficheiros do código-fonte encontram-se na pasta "source_files"
e os ficheiros .h encontram-se na paste "header_files".

Para compilar o mesmo basta correr a batch file chamada "compile_script.bat"
e para correr baster executar a batch file
chamada "run_script.bat".

Os ficheiros "carros.txt", "pilotos.txt" e "autodromos.txt" contém os dados para construir os pilotos, carros e autodromos.

O ficheiro "testes.txt" contém um conjunto de comandos que podem ser colados para testar o programa.
